Теоретически при длине списка n, реализация через synchronizedList будет делать (n−1) шагов каждый проход оп списку, а для точной сортировки понадобится (n−1)^2 шагов.
Реализация через CustomLinkedList будет делать T*(n−1) шагов каждый проход оп списку в худшем случае, где T - число потоков, а для точной сортировки понадобится T*(n−1)^2 шагов.
